# Elevate Labs – Task 4: Dashboard Design (Tableau)

## Objective
Design an interactive dashboard for business stakeholders using Tableau.  
Focus on sales, profit, category trends, segment performance, and geographical insights.

## Deliverables
- Tableau Interactive Dashboard
- PowerPoint Summary Presentation
- Detailed PDF Report with Screenshots + Insights

## Key Insights
- Technology category leads revenue.
- Consumer segment contributes the highest sales.
- California & New York generate maximum profit.
- Texas and Illinois show significant losses.
- Strong seasonal peaks in March, September, and November.
- Profit margin stands at ~12.47%.
